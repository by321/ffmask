FFMask Models Summary

Model Description
U²-Net A deep learning model for salient object detection that accurately identifies and segments the most visually prominent objects in images using a nested U-structure architecture with supervision at multiple scales.
U²-Net-p The "lite" version of U²-Net with reduced parameters and computational complexity, maintaining high-quality salient object detection performance while being more suitable for deployment on resource-constrained devices.
U²-Net-hs A human-segmentation variant of U²-Net specifically trained for detecting and segmenting human figures in images, optimized for applications like photo editing and portrait extraction with high accuracy on human subjects.
DIS Dichotomous Image Segmentation model that uses a fully convolutional neural network with multi-scale feature integration to predict pixel-wise saliency maps, achieving state-of-the-art performance on benchmark datasets for salient object detection.
BiRefNet Lite The lite version of BiRefNet, a state-of-the-art model that employs a novel bilateral reference approach that integrates multi-scale features and attention mechanisms to achieve superior accuracy.
Although more accurate than U²-Net and DIS in general, BiRefNet is also larger and slower, even the lite version of BiRefNet is larger than the full version of U²-Net and DIS.
Face Mask This is implemented using:
  • Haar Cascade, a classical machine learning method using Haar-like features and cascade classifiers for rapid object detection, particularly effective for face detection in OpenCV, known for its speed and efficiency on traditional hardware.
  • Face_recognition package by Adam Geitgey, a Python library built on dlib that provides simple APIs for face detection, facial landmark extraction, and face recognition using deep learning models.
  • Dlib, an open-source C++ library for machine learning, computer vision, and image processing, offering robust tools for shape detection and object recognition, with a Python interface for easy integration in packages like face_recognition.

These models represent different approaches in computer vision from classical ML to modern deep learning techniques.
